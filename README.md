# 🟣 Tiny Dodger

**Tiny Dodger** is a simple dodge-based game where you control a small character and avoid enemies for as long as possible. There is only one level, and the goal is simply to survive.

This project was made as a practice exercise for input handling and collision detection, with added sound effects for a more complete game feel 🎧✨

---

## 🎮 Gameplay

Move your character and avoid all incoming enemies.

There is no score system—just survive as long as you can!

---

## ✨ Features

* Simple dodge-based gameplay
* One endless survival level
* Basic collision detection
* Background music 🎵
* Game over sound effect 🔊

---

## 🕹️ Controls

* **WASD** – Move player
* **Arrow Keys** – Move player

---

## 🔊 Sound

* Background music plays during gameplay
* Game over sound plays when the player is hit

---

## 🚀 Purpose

This project was created to practice and improve:

* Player input handling
* Collision detection
* Basic game loop design
* Simple audio integration

---

## 🧠 Notes

* No scoring system
* No multiple levels
* Focus is purely on survival and movement practice
